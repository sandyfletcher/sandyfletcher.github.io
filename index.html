HTML

In the 1980s, researchers at CERN were working with thousands of complex documents that all referenced each other. One employee, computer scientist Tim Berners-Lee, proposed a system built off hypertext - links that allow you to navigate a network of different pages with ease. He later built off this idea to create Hypertext Markup Language (HTML), the structural backbone of the modern internet.

HTML was designed to be easy for non-technical users to understand, so the layout is essay-like. It starts with a header to introduce basic information about the site, body containing the bulk of the page content, and footer to leave the reader with closing information and signatures. To understand better, let’s make a site of our own.

Like all computer code, HTML for your website is just text. Even pictures are just links written as text. That means you don’t need anything fancy to get started writing your code, just do it in any text editor. If you’re using Windows, you can right-click anywhere on the desktop to bring up an option for a new text file.

Let's copy in some code, and go line by line to understand what it's doing:

<!DOCTYPE HTML>
<html>
<head>
  <title>Your First Website</title>
  <header>
    <h1>Welcome to Websites!</h1>
    <p>A place to experiment with HTML features.</p>
  </header>
</head>
<body>
<p>All killer no filler.</p>
</body>
<footer>
  <footer id="contact">
  <h2>Contact Us</h2>
  <p><a href="mailto:info@yoursite.com">Email</a>!</p>
</footer>
</html>

When you create any kind of computer file, the first thing you do is identify what type of file it is so the computer knows what to expect for the rest of the document. In this case, we lead off by declaring it an HTML file.

Next up is the <header> section, where we set the size of the window by making it one size minus another (extrapolate). We'll also set the title of the page, and add a header section with a title and opening sentence.

From there we'll move into the <body> section.

Adding sections with section IDs

<section id="about">
  <h2>Breaking Things Into Sections</h2>
